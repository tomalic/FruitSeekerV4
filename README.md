# FruitSeeker v2 (PWA)

Mejoras:
- Detección automática de la fila de cabeceras (escanea las primeras 50 filas).
- Banner claro cuando falta la librería XLSX e inclusión de botón **Cargar demo** para probar la búsqueda al instante.
- Cache del Service Worker actualizada (v2) para forzar la actualización.

Para Excel sin internet, añade `lib/xlsx.full.min.js` localmente al proyecto (SheetJS CE). Con **CSV** no necesitas librerías.
